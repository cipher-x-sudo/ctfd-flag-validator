Docker Configuration
  • Docker Type:
    • Local: Uses local socket /var/run/docker.sock.
    • SSH: Remote Docker over SSH (requires Key/Agent).
  • Routing Modes:
    • Host:Port: Uses Connection Hostname + allocated port (e.g. challenges.ctf.com:30001). Used for TCP/SSH.
    • Subdomain: Uses Base Domain (e.g. uuid.challenges.ctf.com). Used for Web/HTTP.
  • Connection Hostname:
    The public domain/IP for Host:Port challenges.
    Security: Do not use the same domain as CTFd to prevent cookie theft.
